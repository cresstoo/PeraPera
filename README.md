# PeraPera - 日本語発音評価アプリ

日本語学習者のための発音評価アプリケーション。Azure Speech Service と MeCab を使用して、発音とグラマーを評価します。

## 特徴

- リアルタイム発音評価
- 文法分析
- 詳細なフィードバック
- 視覚的な進捗表示

## 技術スタック

- Frontend: React + TypeScript
- Backend: Node.js + Express
- Speech Analysis: Azure Speech Service
- Grammar Analysis: MeCab
